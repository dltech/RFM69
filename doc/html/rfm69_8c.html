int rfm69_init ( void  )

This function performs radiomodule initialisation. It writes radiomodule register values via SPI, which were defined and calculated in file rfm69.h. It also checks the SPI interface, if it doesn't work, it sets the RFM69_SPI_FAILED state.

Returns
0 if initialisation completed successfully, and -1 otherwise
int rfm69_transmit_start ( uint8_t  packet_size_loc,
uint8_t  address 
)

Start packet transmittion. This function assumes that radiomodule is in receive, sleep or standby mode. Also do not forget to switch the transmitter off when package is already transmitted (interrupts will do that).

Parameters
packet_size_loc- size of the packet
address- destination address (use broadcast address 0xff if you do not know any)
Returns
negative value if something wrong (package is too big, or SPI doesn't work), and return 0 if all is well
