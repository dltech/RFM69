RFM69 Documentation

This library implements packet data transmittion via RFM69CW radiomodules.

In order to use this library you need to include the rfm69.h file, declare extern global variables and call rfm69_init() function, tat's all.

After init radiomodule will switch into receive mode and will wait the packet by the radio channel. To transmit the packet you need to place your data into packet_buffer and call rfm69_transmit_start() function, then after transmitting data, radiomodule will be switched by interrupt into receive mode again.

If radiomodule received the package successfully, the value of rfm69_condition equals to RFM69_NEW_PACK and received data are alreary in the packet_buffer array. Then, when you are ready to receive new data, you need to start the receive mode again by calling rfm69_receive_start() function.

Note that radiomodule settings are in the rfm69.h file, packet size is in packet_size global variable and the packet data are in packet_buffer global array and radiomodule state is in rfm69_condition variable.
